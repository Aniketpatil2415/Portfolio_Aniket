Movie Recommendation System

PythonMachine LearningStreamlit

About Project

This is an intelligent recommendation engine developed to help users discover movies tailored to their unique tastes. Instead of generic suggestions, the system performs a deep analysis of movie metadata—including genres, keywords, cast, and directors—to find mathematical similarities between films. By leveraging Natural Language Processing (NLP), the application converts textual descriptions into high-dimensional vectors. It then calculates the Cosine Similarity between these vectors to provide high-accuracy suggestions from a dataset of over 5,000 movies. The result is a personalized discovery tool that understands "what" makes a movie similar to another beyond just the title.

Key Features

• Content-Based Filtering: Delivers personalized suggestions by analyzing the core attributes (tags, genres, cast) of the movies you already like.
• NLP Data Processing: Utilizes NLTK and CountVectorizer to clean and process raw metadata into a structured format for the AI model.
• TMDB API Integration: Dynamically fetches and displays high-quality movie posters in real-time to provide a premium, visual browsing experience.
• Vector Space Modeling: Employs advanced mathematical techniques to measure the distance between movie profiles, ensuring top-tier relevance in recommendations.
• Interactive Streamlit UI: A clean, responsive, and user-friendly interface that allows users to search and get results instantly without any lag.
